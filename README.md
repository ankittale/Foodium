The project follows the MVVM (Model-View-ViewModel) architecture and incorporates several new features while embracing an offline-first approach. It relies on the Spoonacular API, allowing users to retrieve and learn various recipes. Users can not only browse through a collection of recipes but also explore the step-by-step process of making each recipe.
